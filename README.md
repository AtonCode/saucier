## Instalacion
Descargar e instalar **node.js**
Lugo instalar **Angular CLI** con el siguente comando:
``` npm install -g @angular/cli ```

Finalmente acceder a la capeta principal de proyecto y ejecutar este comendo:
``` npm install && npm start ```
